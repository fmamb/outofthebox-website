# Charte web - Out of the Box / Signalétique du discernement

## Intention

Créer une identité de dossier d’investigation stratégique : le site doit évoquer une table de montage, une fiche d’audit, une carte de risques et une planche de preuves. Le rendu doit rester premium, sobre et opérationnel.

Promesse visuelle : rendre visible ce que les usages IA ne montrent pas.

## Palette validée par les nouveaux supports

- Orange signal : `#FF7518`
- Bleu nuit : `#242E67`
- Bleu profond : `#171F4C`
- Bleu panneau : `#2F5677`
- Gris interface : `#8E9297`
- Gris clair : `#E9EDF0`
- Fond blanc : `#FFFFFF`
- Fond papier : `#FFFAF2`

## Typographies

- Titres : sans-serif condensée, très grasse, uppercase. Références Canva : Anton, Bebas Neue, Archivo Black, League Spartan.
- Texte : Inter, Montserrat, Source Sans 3 ou Open Sans.
- Labels / preuves / notes : IBM Plex Mono, Space Mono ou une mono compacte.

## Motifs graphiques

- Bandes diagonales orange de signalétique.
- Cadres de visée et coins de cadrage.
- Plaques papier avec léger grain.
- Zones masquées en rectangles noirs.
- Tampons type "Hors-champ identifié".
- Numéros de preuve : `01`, `02`, `Preuves #03`.
- Matrices, checklists, tableaux d’audit.

## Règles UI

- Les sections claires reprennent la logique de plaquette : fond blanc, barre latérale orange, bandeaux bleus, grands titres bleu nuit.
- Les sections sombres restent des zones de projection / investigation.
- Les cartes principales utilisent des panneaux bleus ou blancs avec accents orange.
- Les CTA principaux sont orange, rectangulaires, nets.
- Les hover restent sobres : léger déplacement, ombre jaune, pas d’effet gadget.

## À éviter

- Robots, cerveaux lumineux, hologrammes, violet néon.
- Gradients IA génériques.
- Promesses vagues de productivité.
- Formes arrondies excessives.
- Décorations gratuites sans fonction de lecture.
